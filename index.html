  1. The editor is a part of your project. You start with a default preset and then customize the editor by writing your own project-scoped components. Different experience for each type of project, heavily integrated into your framework (e.g.: AngularIDE, EmberText)
  2. ObjC and Javascript (React Native)., no html and css engines inside. The critical parts written in Objective C using Cocoa TextKit in syncrhonyous way to achieve small latency while typing. Other parts are in Javascript to hackability. Chrome Devtools, hot reloading, ES7.
  3. Open source. Github (coming soon)
  4. OS X only for now. No ecosystem, no plugins yet. Help with that (ping me) or support on Patreon.
