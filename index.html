Skelet CodePoC

Demo

  1. Provokes to build the editor as a part of your own project.
  2. Written in ObjC and Javascript (React Native), no html and css engines inside.
  3. 1MB binary, 200MB RAM.
  4. Github link (not ready)
  5. OSX only, no ecosystem. You can help with that (ping me) or support on Patreon.

1. The editor is also a part of your project

  • a) You start with a preset and global scoped plugins and then customize the editor by writing your own components.
  • b) You can get a totally different expirience for each type of project,
    heavily integrated into your framework (e.g.: AngularIDE, EmberText). React tree component analysis (video)
  • c) Installing plugins as simple as npm install skelet-plugin-react.
  • e) Gradual typing helper, advanced debugging.
  • d) Add bootstraper to your Editor (webpack, hotreload).

2. Arhitecture

  • a) It's a React Native app.
  • b) UIThread - JSThreads illustration.
  • c) Chrome DevTools screenshot.
  • d) Hot reloadable.
  • e) ES7, JSX.

3. Perfomance

  • Native Cocoa TextKit under the hood.
  • Typing latency ~x ms.
  • Allows you to open 1GB file
